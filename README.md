# 🚀 Startup CLI

Startup CLI es una herramienta de línea de comandos diseñada para **organizar, crear y administrar múltiples proyectos tecnológicos desde una sola estructura de desarrollo**.

Permite a equipos o founders mantener **orden, consistencia y velocidad** al trabajar con múltiples productos.

Ideal para:

- startups con varios productos
- equipos full-stack
- proyectos con frontend, backend y cloud
- entornos con múltiples repositorios

---

# 🎯 Qué hace Startup CLI

Startup CLI automatiza la creación y organización de proyectos.

Permite:

- crear proyectos con estructura estándar
- organizar frontend, backend y base de datos
- centralizar documentación
- navegar entre proyectos
- mantener arquitectura consistente
- facilitar deploy y mantenimiento
